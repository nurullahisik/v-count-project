# Installation
## 1. Copy environment file

    cp .env.example .env

or

    change .env.example file to .env 

**Note: .env file must be filled for database connection**

## 2. Install Packages
    
    npm install

## 3. Run Project
    
    node app.js

or (With Nodemon)

    npm run start
